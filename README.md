# Irfan View'er
Remote for Unified Remote for viewing photos

Features:

- Folder Browser With Navigation
- Full Screen
- Manual Previous - Next Image
- Rotate Left - Right
- Zoom Out - In
- Zoom Navigation
- Flip Horizontal - Mirror View
- Random Image In The Folder
- Darken - Brighten
- View In Greyscale
- Restore Original View
- No Delete File Button
